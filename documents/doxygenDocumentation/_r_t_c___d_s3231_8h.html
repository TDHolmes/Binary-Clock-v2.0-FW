◆ RTC_init()

retval_t RTC_init ( rtc_time_t *  t_ptr,
volatile uint8_t *  RTC_int_cnt_ptr 
)

Real time clock (RTC) initialization function that sets up the 1 Hz pin change interrupt and the I2C communication to the chip to configure it and read the time it has internally.

Parameters
[in]time_ptr(rtc_time_t *): Pointer that is used to keep track of the time.
[in]RTC_int_cnt_ptr(volatile uint8_t *): RTC interrupt count pointer so we can keep track of the interrupt count from outside of this file.
[out]retval(retval_t): Returns the initialization status of the RTC chip.
